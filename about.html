About Me

My name is Mustafa Kapadia, I am currently pursuing BScIT from VSIT (Vidyalankar School of Information Technology), Wadala. I got my first computer at age 9 and since then I have been curious about technology surrounding it, from hardware to software.

In Sem 2 of my degree I was exposed to Web Development and since then I have read a lot about the technologies that go into Web Development like HTML, CSS and JavaScript along with languages like PHP and Python that form it’s backbone. When practising HTML and CSS I came across Figma and it was then that I started learning about UI /UX Designs and it’s significance.

So here I am exploring the ocean of Web Development drop by drop.
